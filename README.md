# Study Hours vs Score Predictor

## Overview
A machine learning project that predicts exam scores based on study hours using Linear Regression.

## Features
- Train a linear regression model
- Predict scores for given study hours
- Visualize training data and prediction line
- Model evaluation metrics

## Tech Stack
- Python 3.x
- Scikit-learn (ML framework)
- Pandas (Data manipulation)
- Matplotlib (Visualization)
